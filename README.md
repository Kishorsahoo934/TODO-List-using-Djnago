# 📝 TODO List Web App using Django & PostgreSQL

## 🔥 A Full Stack Project for Daily Task Management

A simple yet powerful **Full Stack TODO List Web Application** built with **Django** and **PostgreSQL**, allowing users to manage their daily tasks efficiently.

---

# 🚀 Features

- 🔐 User Registration & Login  
- ✅ Add, Edit, Mark Complete, and Delete Tasks  
- 📋 Task Filtering (Completed / Pending)  
- 🌐 Responsive UI using Django Templates  
- 💾 PostgreSQL as backend database  
- ⚙️ Admin panel for task management

---

# 🛠️ Tech Stack

| Backend     | Frontend        | Database     | Tools                    |
|-------------|------------------|--------------|---------------------------|
| Django      | HTML, CSS        | PostgreSQL   | Git, GitHub, Python-dotenv |
| Django ORM  | Bootstrap (opt.) | psycopg2     | VS Code, Terminal        |
